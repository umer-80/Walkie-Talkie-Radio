║                    🚀 HOW TO TEST NOW                          ║
╚════════════════════════════════════════════════════════════════╝

STEP 1: Open a web server (choose one):
   
   Option A - Python (recommended):
   $ python3 -m http.server 8080
   
   Option B - Node.js:
   $ npx http-server -p 8080
   
   Option C - Direct file (might have permission issues):
   $ open index.html


STEP 2: Open TWO browser windows/tabs:

   Window 1 (Creator): http://localhost:8080/index.html
   Window 2 (Joiner):  http://localhost:8080/index.html (incognito/private)


STEP 3: Test the connection:

   ┌─────────────────────────────────────────────────────────┐
   │ WINDOW 1 (Creator)                                      │
   ├─────────────────────────────────────────────────────────┤
   │ 1. Click "📻 CREATE CHANNEL"                            │
   │ 2. You'll see a code like "ALFA12" or "KILO7"          │
   │ 3. Copy this code                                        │
   │ 4. Wait for status: "Waiting for partner to join..."   │
   └─────────────────────────────────────────────────────────┘

   ┌─────────────────────────────────────────────────────────┐
   │ WINDOW 2 (Joiner)                                       │
   ├─────────────────────────────────────────────────────────┤
   │ 1. Paste the code from Window 1                         │
   │ 2. Click "JOIN"                                          │
   │ 3. Wait for status: "Connected! Hold PTT to talk"      │
   └─────────────────────────────────────────────────────────┘


STEP 4: Talk to yourself! 🎙️

   In WINDOW 1:
   → Hold down the red PTT button
   → Speak: "Testing 1, 2, 3"
   → Release PTT (you'll hear a beep)
   
   In WINDOW 2:
   → You should hear your own voice!
   → Green RX LED should light up
   
   Switch roles and test the other direction.


╔════════════════════════════════════════════════════════════════╗
║                   🧪 TROUBLESHOOTING                           ║
╚════════════════════════════════════════════════════════════════╝

❌ "Not connected to signaling server"
   → Server might have stopped. Restart with:
   $ node server.js

❌ "Microphone permission denied"
   → Click the 🔒 lock icon in browser address bar
   → Allow microphone
   → Refresh the page

❌ "No audio in other window"
   → Both windows MUST grant microphone permission
   → Hold PTT button WHILE talking (don't click and release)
   → Check TX LED (green) lights up when talking
   → Check RX LED (yellow) lights up when receiving

❌ "Connection failed"
   → Try Chrome or Edge (best WebRTC support)
   → Disable VPN if using one
   → Check firewall settings
